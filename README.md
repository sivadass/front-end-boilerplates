# Gulp Task
Gulp JS setup for minifying and bundling JS and SCSS files

## Installation
Goto the project folder and install required dependencies:
```
npm install
```

And run Gulp to watch for code changes and generate **script.min.js** and **styles.min.css**:
```
gulp watch
```
